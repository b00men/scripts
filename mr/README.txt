Скрипты-костыли для МаркетРадио

auto_query/ - Скрипты автоматизации удаленных процедур над работающими машинками.
auto_query/reload.sh - Обновление по перечислению.
auto_query/reload_list.sh - Обновление для группы или всех. 
auto_query/query_list.sh - Массовое исполнение пользовательских комманд для группы или всех.
auto_query/query.txt - Примеры пользовательских комманд.
auto_query/to_new_tr.sh - Перевод большого колличества машин на новый трафик.

new_traffic.sh - Удаленное разворачивание нового трафика на старом трафике.
instal_new_traffic_default.sh - Старый плохой скрипт установки нового трафика на голую машину.

auto_install/ - Скрипты автоустановки и дорабатывания машинки сразу после установки debian (старый трафик).
auto_install/sshrooton.exp - expect скрипт, включает вход по руту и добавляет ключ.
auto_install/client_old_dexp_install.sh - Запускаемый на клиентской машине скрипт, завершающий установку.
auto_install/install.sh - Оболочка для предыдущих двух скриптов и прочая установка.

clean_stable.sh - Очистка /work (старый трафик, костыль).

whatos.sh - Узнать текущую систему.
ping.sh - Быстрый анализ локальной подсети и mac'ов в ней (для CentOS).
install_videoplayer.sh - Сашин скрипт по установке видеоплеера.

air-nsk/ - Скрипты ускорения создания машин для ссылок и их конфигурации
air-nsk/add_virt.sh - Добавить ссылку по образу cs
air-nsk/virt_config.sh - Частичная конфигурация новой ссылки (сыро)
